Q:В default-originate, субкомманда match-default - в чем предназначение?
A:default-route-advertise [ always | match default
always

Configures an IS-IS device to unconditionally advertise default routes with itself as the next hop.

-

match default

Advertises a default route through an LSP if the default route is generated by other routing protocols or other IS-IS processes in the routing table. If this default route is deleted from the routing table, it is not advertised through an LSP.
